Porter primary action button — solid brand color, no shadow; use it for the one main action on a surface.

```tsx
<Button variant="purple" size="lg">Start free trial</Button>
<Button variant="outline">See templates</Button>
```

Variants: `purple` (primary) · `pink` · `aqua` · `yellow` · `dark` · `outline`. Sizes: `sm` · `md` · `lg`. One primary purple button per decision area; everything else is `outline` or a supporting color.
