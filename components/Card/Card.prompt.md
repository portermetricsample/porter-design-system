Porter content surface — rounded, generously padded, solid background, no drop shadow.

```tsx
<Card background="cream">
  <Eyebrow>Marketing reports</Eyebrow>
  <Heading level="h2">Your data, <strong>on brand</strong></Heading>
</Card>
```

Backgrounds: `white` · `cream` · `lavender` · `dark` · `purple`. Use `bordered` for a hairline on light surfaces. Pair `dark`/`purple` with light text. Max 3 colors per card.
